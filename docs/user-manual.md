## 1. Hyrje
Ky manual shpjegon përdorimin bazë të platformës ProEstate.

## 2. Regjistrimi
1. Hap faqen kryesore.
2. Kliko `Register`.
3. Plotëso emrin, mbiemrin, email-in, fjalëkalimin dhe rolin.
4. Kliko `Create Account`.
5. Kontrollo email-in për konfirmim nëse aktivizohet email verification.

## 3. Login
1. Kliko `Login`.
2. Vendos email-in dhe fjalëkalimin.
3. Kliko `Sign in`.
4. Pas login-it hapet dashboard-i.

## 4. Shtimi i një prone
1. Hyr si Agent ose Owner.
2. Shko tek `My Properties`.
3. Kliko `Add Property`.
4. Plotëso titullin, përshkrimin, çmimin, qytetin, adresën, tipin, dhomat dhe sipërfaqen.
5. Ngarko foto dhe dokumente.
6. Kliko `Save`.

## 5. Kërkimi i pronave
1. Hyr si klient ose përdorues i regjistruar.
2. Shko tek faqja `Properties`.
3. Përdor filtrat: qytet, çmim, tip, dhoma, sipërfaqe.
4. Kliko `Search`.
5. Hap detajet e pronës që të intereson.

## 6. Rezervimi i takimit
1. Hap detajet e pronës.
2. Kliko `Book Appointment`.
3. Zgjidh datën dhe orën.
4. Shkruaj mesazh opsional.
5. Kliko `Submit`.
6. Prit konfirmimin nga agjenti/pronari.

## 7. Favorites
1. Hap një pronë.
2. Kliko ikonën `Favorite`.
3. Prona ruhet në listën tënde të preferuar.

## 8. Admin Panel
Admin mund të:
- Shikojë përdoruesit
- Menaxhojë pronat
- Shikojë statistika
- Moderojë përmbajtjen
